Aufgaben sollen nach Status in einem Kanban Board angezeigt werden können.

Für jeden Status-Wert gibt es eine Spalte, deren Header so eingefärbt ist wie der Status-Wert.

Aufgaben sollen natürlich zwischen den Spalten verschoben werden können (was eine Statusänderung bewirkt).

Aufgaben sollen auch innerhalb einer Spalte vertikal verschoben werden können. Überlege dir, wie du das intern vermerkst.

Karten kann man im Board anlegen, ändern und löschen (zeige das durch Icons an).

Ein Verschieben nach Done entspricht einem Abhaken.
